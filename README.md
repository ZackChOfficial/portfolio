## Next.js Portfolio
A personal portfolio website built with [Next.js](https://nextjs.org/docs)

## Demo
[Live Demo](https://zchatouane.vercel.app/)

## Technologies
 - Next.js: ^13.4.7
 - React: ^18.2.0
 - TypeScript: ^5.1.3
 - Tailwind CSS: ^3.3.2
 - Framer Motion: ^2.3.0

## Getting Started
### Prerequisites
 - Node.js (v14+)
 - npm or Yarn

## Installation

```bash
git clone https://github.com/ZackChOfficial/portfolio.git
cd portfolio
npm install
```

### Running the Project

```bash
npm run dev
```

Visit [http://localhost:3000](http://localhost:3000) to view the project.
